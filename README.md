# Miranda Clinics Queue Management System

A modern, responsive queue management system for Miranda Clinics built with Angular.

## Features

- Responsive public queue display
- Live "Now Serving" status
- Multiple consultation rooms
- Doctor and room information
- Mobile-friendly interface
- Login page for administrative access (coming soon)

## Tech Stack

- Angular
- HTML5
- CSS3
- TypeScript

## Planned Features

- Admin dashboard
- User authentication
- Queue number management
- Dynamic room management
- Real-time updates
- Patient recall and skip
- Queue history
- Docker deployment

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
ng serve
```

Open your browser at:

```
http://localhost:4200
```

## Build

```bash
ng build
```
